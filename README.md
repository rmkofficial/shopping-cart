
# Shopping Cart

This is a shopping cart project built using React and the useContext hook. It allows users to browse products and add them to their shopping carts.


## Features:

- Product listing
- Add to cart
- Cart preview
- Remove items

  
## Techologies Used:

**React**

**useContext** 

**Tailwind.css** 

  
## Project Structure :



- src
    - components
        - Cart.jsx
        - CartIcon.jsx
        - Header.jsx
        - ProductItem.jsx
        - Product.jsx
    - context
        - CartContext.jsx
    - app.jsx
    - index.jsx
    - main.jsx
